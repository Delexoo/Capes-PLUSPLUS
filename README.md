## About

**Capes++** is a Windows desktop utility for **Minecraft Java Edition** that gives players access to iconic and exclusive cape styles. Built for the **1.8.9** client and designed for players who already own at least one free official Minecraft cape (Migrator, Twitch, Mojang Office, Cherry Blossom, 15th Anniversary, and similar).

> **Official releases and support:** [Delexoo on Beacons](https://shop.beacons.ai/delexo/3f484dc8-435a-4a40-9be2-57b53c493c30)

---

## Requirements

| Requirement | Details |
|-------------|---------|
| **Operating system** | Windows only |
| **Edition** | Minecraft **Java Edition** |
| **Game version** | **1.8.9** |
| **Cape eligibility** | At least one free official Minecraft cape on your account |
